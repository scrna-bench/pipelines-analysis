This folder can be populated in two ways:

1. **Run the benchmark locally** — follow the instructions in [pipelines-plan](https://github.com/scrna-bench/pipelines-plan) and copy the `out/` folder into this directory.

2. **Download from Zenodo** — download the archive from [https://doi.org/10.5281/zenodo.18848104](https://doi.org/10.5281/zenodo.18848104), unzip it, and copy the `out/` folder into this directory.

The `billato.csv` file was compiled from Table 1 of [Billato et al. (2025)](https://www.biorxiv.org/content/10.1101/2025.10.28.681564v1).
